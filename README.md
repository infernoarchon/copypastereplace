# Copy Paste Replace

A word replacement game. Users can create funny stories using OMDB, Google Books API, or paste in their own text content and our algorithm will analyze and parse the text for replacement. 

## Setup

1. `npm install`
2. Set API keys in `.env` file
3. Download and install MongoDB
4. Run `mongod` to start MongoDB locally
5. `npm start`

## APIs

- Google Natural Language
- Google Text to Speech
- Google Books
- OMDB
- Wordnet
- GIPHY

## Technologies

- MERN (MongoDB, Express, React, Node)
- Passportjs
- Axios
- Momentjs
- Json-query
- Mongoose
